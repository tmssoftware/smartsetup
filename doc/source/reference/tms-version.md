# tms version

Prints the current version of the tms utility.

## Synopsis

```shell
tms version [<global-options>]
```

## Description

Prints the version number of the installed `tms` executable to the console. The output has the form:

```
tms version <version-number>
```

## Global Options

See [Global Options](xref:SmartSetup.Command.GlobalOptions) for options available to all commands.

## Examples

Displays the version of the installed tms tool, for example `tms version 3.1`:

```shell
tms version
```
